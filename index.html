Latage v0.8

Tool creato da Rosario Terranova per la conversione real-time di formule Latex in immagini e url usando le Google chart API e Query string encoder. Puo' anche calcolare il risultato trasferendo i dati al servizio Wolfram Alpha. Latage e' un software open source, puoi scaricare il codice sorgente qui.
